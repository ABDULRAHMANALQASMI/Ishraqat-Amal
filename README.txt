README - منصة جمع طلبات المساعدة (نسخة جاهزة للنشر)

ماذا تحتوي الحزمة؟
- index.html    -> صفحة عامة للمستخدمين لتقديم طلبات
- admin.html    -> لوحة مشرف بسيطة لعرض الطلبات
- styles.css    -> أنماط CSS
- app.js        -> منطق الواجهة لرفع الطلبات للـ Firebase
- admin.js      -> منطق لوحة المشرف
- firebase-config.js -> ملف تحتاجين تملئينه بإعدادات مشروع Firebase
- assets/       -> أيقونات/صور بديلة

خطوات إعداد Firebase (موجز):
1. افتحي https://console.firebase.google.com وأنشئي مشروع جديد.
2. فعّلي Firestore (Native mode).
3. فعّلي Firebase Storage.
4. فعّلي Authentication > Sign-in method > Email/Password.
5. في إعدادات المشروع > عام > "أضف تطبيق ويب" وانسخي إعدادات الـ firebase config إلى ملف `firebase-config.js`.
6. اذهبي إلى قواعد Firestore وStorage واضبطيها حسب مستوى الأمان المطلوب. للأختبار يمكنك وضع قواعد تسمح بالقراءة والكتابة للمستخدمين المسجلين فقط، أو تقييد الوصول إلى admins.

نقاط أمان مهمّة:
- تأكدي من إعداد قواعد Storage بحيث لا تكون عامة (أو حددي مجلدات خاصة بالطلبات).
- يجب فقط منح صلاحيات لوحة المشرف لمستخدمين موثوقين (سجلي حساب مشرف عبر Authentication).
- لأجل الإنتاج، فكري بإضافة مراجعة بشرية قبل الموافقة على نشر بيانات أو مشاركة ملفات حساسة.

نشر الموقع:
- بإمكانك رفع الملفات إلى GitHub Pages أو Netlify أو Vercel.
- ستحتاجين لتعديل `firebase-config.js` بقيم مشروعك.

مساعدة إضافية:
- أقدر أعدّل القواعد الأمنية، أو أعمل صفحة تسجيل للمتطوعين أو نظام تتبع حالة الطلب (موافق، قيد المراجعة، مكتمل).
- لو تبي، أقدر أرفع لك نسخة ZIP جاهزة الآن أو أعدّل الألوان والشعار.
